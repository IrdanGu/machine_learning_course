# Machine Learning A-Z (Codes and Datasets)

This repository contains my progress and code implementations for the **Machine Learning A-Z™: AI, Python & R + ChatGPT Bonus [2025]** course. It covers various aspects of machine learning from data preprocessing to advanced algorithms.

## 🚀 Progress Summary

### Part 1: Data Preprocessing
Completed the foundational steps for preparing datasets for machine learning models.
- **Importing Libraries**: Setting up the environment with `numpy`, `matplotlib`, and `pandas`.
- **Importing Dataset**: Loading data and separating independent (`X`) and dependent (`y`) variables.
- **Handling Missing Data**: Using `SimpleImputer` to fill missing values with the mean.
- **Encoding Categorical Data**:
    - **Independent Variables**: Using `OneHotEncoder` and `ColumnTransformer`.
    - **Dependent Variables**: Using `LabelEncoder`.
- **Splitting Dataset**: Dividing the data into **Training Set** (80%) and **Test Set** (20%).
- **Feature Scaling**: Applying `StandardScaler` to normalize the range of independent variables.
